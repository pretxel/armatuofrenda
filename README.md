armatuofrenda
=============

Aplicación Web para el día festivo en México "Día de Muertos", consiste en crear tu ofrenda digital y guardarla para compartirla con tus seres queridos.

#Herramientas:
* HTML5
* jQuery (Drag and Drop)
* html2canvas.js
* touch-punch.js
* video.js.
